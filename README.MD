# libmeta

Just for fun and learning.
A c++ runtime reflection library.

## Dependencies

- c++20 compiler
- libclang
- cmake

## Third part libraries

- [nlohmann/json](https://github.com/nlohmann/json)
- [pugixml](https://github.com/zeux/pugixml)

## Credits
